## Excel Sales Analysis Project

## Project Overview

This project analyzes sales performance for Okenwa Electronics Ltd using Microsoft Excel.

The goal was to identify key sales trends, top-performing products, regions, and salespeople while presenting insights through an interactive dashboard.

## Business Questions

1. What is the total sales revenue?
2. What is the total profit?
3. Which product generated the highest sales?
4. Which region generated the highest sales?
5. Which salesperson generated the highest revenue?

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* SUM
* AVERAGE
* IF
* COUNTIF
* SUMIF

## Key Findings

* North region generated the highest sales.
* Laptops generated the highest revenue.
* Total sales amounted to ₦6,845,000.
* The company recorded a total profit of ₦1,730,000.
